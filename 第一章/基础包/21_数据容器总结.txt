# 是否支持下标索引
-支持 : 列表,元组,字符串   ->序列类型
-不支持 : 集合,字典   ->非序列类型

# 是否支持重复元素
-支持 : 列表,元组,字符串
-不支持 : 集合,字典

# 是否可以修改
-支持 : 列表,集合,字典
-不支持 : 元组,字符串

列表 :可修改可重复的一批数据记录场景
元组 :不可修改可重复的一批数据记录场景
字符 :一串字符串的存储场景
集合 :去重存储的场景
字典 :利用key检索value存储的场景

# 遍历  尽管形式不同但是都支持遍历操作
for循环
    五类容器都支持
while循环
    只有列表元组字符串支持,因为字典,集合不支持下标索引

# 通用函数
len(容器) :容器长度
max(容器) :容器内最大元素
min(容器) :容器内最小元素

list(容器) --转为列表
str(容器) --转为字符串  转完实际是带双引号
tuple(容器) --转为元组
set(容器) --转为集合

# 数据容器间的比较
sorted(容器,[reverse=True])  --将给定容器进行排序,第二参数默认是false,传入reverse=True可以反向排序

# 拓展 字符串的对比是依靠ascii对比的
